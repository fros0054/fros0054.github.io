HTML Tags That I've Learned

Header Tag
The Header tag acts as the big title at the top of the page, but when used inside of an Article tag, it becomes the title of that article. There are 6 header tags from 1-6, the lower the number the bigger the text will be, allowing you to add different size of headers for any situation.
Footer Tag
The Footer tag is used as the very bottom text of a page, it is normally used to add a copyright tag to a page. The Footer tag can also be used inside an Article tag, to be used as the Footer for that article.
Sub Tag
Adding the Sub is very situatinal, it's used to add subscript to a word, for exaple ~ H2O.
Cite Tag
Using the Cite tag allows you to add a citation for another source. These are primarily used to cite another person's name, a quote from another person, so on and so forth.
Mark Tag
This tag is quite fun use, it allows you to add a highlight to a specific piece of text. This is mainly used to put a highlight on specific keywords on a results page after the user has serched for something.
